## 1. DuckDuckGo Search API (Unofficial)
**URL:** https://api.duckduckgo.com/

**Pros:**
- No API key required
- Instant answers, summaries, infoboxes

**Cons:**
- Limited to instant answers (not full search results)
- No ranked link list like Google

---

## 2. Brave Search API (Free Tier)
**URL:** https://brave.com/search/api/

**Pros:**
- Privacy-focused
- 200 free searches/day

**Cons:**
- Requires free API key
- No autocomplete or advanced search filters on free tier

---

## 3. Bing Web Search API (via Microsoft Azure)
**URL:** https://portal.azure.com

**Pros:**
- Free tier includes 1,000 requests/month
- Full web search results

**Cons:**
- Requires Azure account and setup
- API key required

---

## 4. Google Programmable Search Engine (CSE)
**URL:** https://programmablesearchengine.google.com/

**Pros:**
- Uses Google search results
- Free and customizable

**Cons:**
- Limited to specific sites unless configured broadly
- Requires some setup with Google

---

## 5. ContextualWeb Search API
**URL:** https://rapidapi.com/contextualwebsearch/api/web-search

**Pros:**
- 10,000 free requests/month via RapidAPI
- Returns structured results

**Cons:**
- Hosted via RapidAPI (may limit performance)
- Less reliable than Google or Bing
